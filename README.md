## Description

KNS Cocktail Finder is a web-based application designed to help users easily
discover and explore a wide variety of cocktail recipes. The application
provides an interactive and user-friendly solution for discovering new
cocktails and managing personalized drink collections.

Done by:

- Kai Chin Aleong (UI):

I was responsible for designing the user interface of the KNS Cocktail Finder application. I began by creating wireframes for key pages, including the home page, login page, dashboard, and cocktail details page. These wireframes helped define the overall layout and structure of the application before development began.

I selected an appropriate colour palette and typography to ensure a visually appealing and consistent design across all pages. I then implemented the interface using HTML, CSS, and Bootstrap, focusing on delivering a clean, modern, and user-friendly experience.

I implemented Progressive Web App (PWA) functionality by creating a manifest.json file and configuring a service worker to enable offline support through caching of core application files.
